Pasos para crear un repositorio y subirlo a github

-crear un repositorio en el directorio donde estan los ficheros y/o carpetas
	git ini nombre_del_repositorio

-entrar en el directorio donde se encuentra el repositorio y añadir ficheros al directorio
	cd nombre_del_repositorio
	git add fichero

-hacer un commit en local
	git commit -m "descripción del commit"

-subir a github
	seguir los pasos que se muestran en la web de github


	

se puede comprobar el estado del repositorio desde el propio directorio usando
	git status
